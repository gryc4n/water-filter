## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/gryc4n/water-filter.git
```

### Navigate to the project folder

```bash
cd hydroflow-landing-page
```

### Open the project

Simply open:

```text
index.html
```

in your browser.
